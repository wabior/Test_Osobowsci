# Test_Osobowsci
Projekt wykonany w ramach kursu Inżynierii Oprogramowania dla AHE Łódź, wydział Świdnica

Główny folder 'Test_Osobowości' to projekt Pycharma w Pythonie.
Podfolder 'oso' zawiera projekt Django.

Aby uruchomić program, musi być zainstalowyany Python. Wskazujemy Pycharmowi forder projektu Pycharm.
W konsoli, będąc w katalogu 'oso' zawierającym plik 'manage.py', wydajemy komendę: 'python manage.py runserver'
